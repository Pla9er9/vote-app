## Run Locally

Clone the project

```bash
  git clone https://github.com/Pla9er9/vote-app
```

Go to the project directory

```bash
  cd vote-app
```

Install dependencies

```bash
  npm install
```

Create .env file
```env
TWITCH_CLIENT_ID = 
TWITCH_CLIENT_SECRET = 
MONGODB_USERNAME = 
MONGODB_PASSWORD = 
MONGODB_URL = 
NEXTAUTH_SECRET = 
NEXTAUTH_URL = 
```

Start the server

```bash
  npm run dev
```
